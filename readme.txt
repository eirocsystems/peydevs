== Description ==

PeyDevs is a WordPress plugin that adds functionality to remind clients to pay their developers. It automatically sends reminders to clients about pending developer payments and allows for customization of reminder intervals and messages.

Main features:

- Automatic reminders to clients to pay developer by the website content disappearing after at the set date.
- Customizable dates.
- Option to activate and deactivate.
- Support to the latest WordPress theme.

For more information, visit [Plugin Homepage](https://amadratitus.github.io/peydev/).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/peydevs` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings under 'PeyDevs' in the WordPress admin menu.
